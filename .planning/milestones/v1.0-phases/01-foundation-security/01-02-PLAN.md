   - Define `CSM_ROOT` -- BUT do not resolve here. CSM_ROOT is set by the entry point (bin/csm). Instead, add a guard: `[[ -n "${CSM_ROOT:-}" ]] || { echo "ERROR: CSM_ROOT not set. Run via bin/csm." >&2; exit 1; }`
   - Color helpers using ANSI codes (not tput, for simplicity):
     - `msg_info()` -- blue prefix "[*]"
     - `msg_ok()` -- green prefix "[OK]"
     - `msg_warn()` -- yellow prefix "[!]"

   - `platform_check_dependencies()`:
     - Verify `docker` is in PATH, else die
     - Verify `jq` is in PATH, else die
     - Verify `ssh-keygen` is in PATH, else die
     - Verify Docker daemon is running: `docker info &>/dev/null || die "Docker is not running"`
